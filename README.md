# Song Cleaner

A [Geode](https://geode-sdk.org/) mod for Geometry Dash that automatically deletes a level's downloaded song when you remove the level — so unused `.mp3` files don't pile up in your `GeometryDash` AppData folder.

See [about.md](about.md) for the user-facing description.

## Building

Requires the [Geode SDK](https://docs.geode-sdk.org/) with `GEODE_SDK` set.

```bat
build.bat
```

The packaged mod ends up at `build/exilium.song-cleaner.geode`.
